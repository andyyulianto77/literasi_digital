---

### **1. CSS di dalam `HAXCMSGlobalStyleSheetContent`**
- **Cakupan (Scope):**
  - Gaya yang didefinisikan di dalam `HAXCMSGlobalStyleSheetContent` diterapkan secara **global** ke seluruh dokumen HTML, termasuk elemen di luar komponen.
  - Gaya ini memengaruhi elemen di tingkat **global**, seperti elemen `<body>`, `<html>`, atau elemen lain yang tidak berada di dalam Shadow DOM.

- **Tujuan:**
  - Digunakan untuk mengatur gaya global yang memengaruhi seluruh halaman atau aplikasi.
  - Biasanya digunakan untuk mengatur tema global, seperti warna latar belakang, margin, padding, atau font default.

- **Contoh:**
  ```javascript
  HAXCMSGlobalStyleSheetContent() {
    return [
      ...super.HAXCMSGlobalStyleSheetContent(),
      css`
        :root {
          --my-theme-low-tone: var(--ddd-theme-default-slateMaxLight);
          --my-theme-high-tone: var(--ddd-theme-default-coalyGray);
        }
        body {
          padding: var(--ddd-spacing-0);
          margin: var(--ddd-spacing-0);
          background-color: var(--my-theme-low-tone);
        }
        body.dark-mode {
          background-color: var(--my-theme-high-tone);
        }
      `,
    ];
  }
  ```
  - Gaya ini memengaruhi elemen `<body>` di seluruh dokumen, termasuk elemen yang tidak berada di dalam komponen.

---

- **Cakupan (Scope):**
  - Gaya yang didefinisikan di dalam `super.styles` hanya diterapkan pada elemen **host** (komponen) dan elemen yang berada di dalam **Shadow DOM** dari komponen tersebut.
  - Gaya ini **tidak memengaruhi elemen di luar komponen**.

- **Tujuan:**
  - Digunakan untuk mengatur gaya khusus untuk komponen tertentu.
  - Biasanya digunakan untuk mengatur tata letak, warna, atau gaya elemen internal dari komponen.

- **Contoh:**

      super.styles,
      css`

        }
      `,
    ];

  ```

---

### **Perbedaan Utama**
| **Aspek**                | **HAXCMSGlobalStyleSheetContent**                              | **super.styles**                                   |
|--------------------------|---------------------------------------------------------------|--------------------------------------------------|
| **Cakupan (Scope)**       | Global (seluruh dokumen HTML).                                | Lokal (hanya di dalam komponen dan Shadow DOM).  |
| **Tujuan**                | Mengatur gaya global untuk seluruh halaman atau aplikasi.     | Mengatur gaya khusus untuk komponen tertentu.    |
| **Pengaruh pada Shadow DOM** | Tidak memengaruhi elemen di dalam Shadow DOM.                | Hanya memengaruhi elemen di dalam Shadow DOM.    |
| **Penggunaan Umum**       | Tema global, seperti warna latar belakang atau font default.  | Gaya internal komponen, seperti tata letak atau warna elemen. |

- Gunakan **`HAXCMSGlobalStyleSheetContent`** untuk mendefinisikan gaya global yang memengaruhi seluruh dokumen.
- Gunakan **`super.styles`** untuk mendefinisikan gaya lokal yang hanya berlaku di dalam komponen tertentu.

GitHub Copilot: CSS rule berikut:

```css
element {
  all: unset;
}
```

tidak berfungsi karena selector `element` tidak valid dalam CSS. Selector `element` tidak merujuk ke elemen HTML apa pun. Dalam CSS, selector harus merujuk ke elemen HTML yang valid (seperti `div`, `p`, `button`, dll.), class (seperti `.my-class`), id (seperti `#my-id`), atau pseudo-class/pseudo-element.

---

### **Mengapa Tidak Berfungsi?**
1. **`element` Bukan Selector yang Valid:**
   - CSS tidak mengenali `element` sebagai selector yang valid. Jika Anda ingin menargetkan semua elemen, gunakan selector universal `*`.

2. **`all: unset;` Tidak Diterapkan:**
   - Bahkan jika selector valid, properti `all: unset;` hanya akan diterapkan pada elemen yang cocok dengan selector tersebut. Karena `element` tidak valid, aturan ini tidak diterapkan ke elemen mana pun.

---

### **Cara Memperbaiki**
Jika tujuan Anda adalah menghapus semua gaya default dari elemen tertentu atau semua elemen, gunakan salah satu pendekatan berikut:

---

#### **1. Untuk Semua Elemen**
Gunakan selector universal `*` untuk menargetkan semua elemen di dalam cakupan CSS.

**Contoh:**
```css
* {
  all: unset;
}
```

**Catatan:**
- Ini akan menghapus semua gaya default dari semua elemen, termasuk elemen penting seperti `<html>` dan `<body>`. Gunakan dengan hati-hati karena dapat menyebabkan elemen kehilangan gaya dasar seperti margin, padding, atau font.

---

#### **2. Untuk Elemen Tertentu**
Jika Anda ingin menargetkan elemen tertentu, gunakan selector yang sesuai, seperti `div`, `button`, atau elemen lainnya.

**Contoh:**
```css
button {
  all: unset;
}
```

---

#### **3. Untuk Elemen di dalam Komponen**
Jika Anda ingin menargetkan elemen di dalam komponen tertentu, gunakan selector berbasis `:host` atau elemen spesifik.

**Contoh:**
```css
:host * {
  all: unset;
}
```

---

### **Catatan tentang `all: unset;`**
- Properti `all` adalah shorthand CSS yang mengatur semua properti yang dapat diwariskan (`inherit`) atau properti yang tidak diwariskan (`initial`) ke nilai defaultnya.
- `unset` berarti:
  - Untuk properti yang dapat diwariskan, gunakan nilai `inherit`.
  - Untuk properti yang tidak diwariskan, gunakan nilai `initial`.

---

### **Kesimpulan**
- Selector `element` tidak valid dalam CSS. Gunakan selector yang valid seperti `*`, `div`, `button`, atau lainnya.
- Jika Anda ingin menghapus semua gaya default, gunakan `* { all: unset; }`, tetapi gunakan dengan hati-hati karena ini dapat memengaruhi seluruh dokumen.
- Jika Anda hanya ingin menargetkan elemen tertentu, gunakan selector yang lebih spesifik.

### **1. Variabel CSS (`var()`)**
- **`var()`** adalah fungsi CSS yang digunakan untuk mengambil nilai dari variabel CSS yang telah dideklarasikan sebelumnya.
- Sintaks:
  ```css
  var(--nama-variabel, nilai-default)
  ```
  - **`--nama-variabel`**: Nama variabel CSS yang ingin digunakan.
  - **`nilai-default`** (opsional): Nilai fallback yang digunakan jika variabel tidak didefinisikan.

---

### **2. Analisis Properti `padding`**
```css
padding: 0
  var(
    --a11y-collapse-padding-right,
    var(--a11y-collapse-horizontal-padding, var(--ddd-spacing-4))
  )
  0
  var(
    --a11y-collapse-padding-left,
    var(--a11y-collapse-horizontal-padding, var(--ddd-spacing-4))
  );
```

- **`padding`** memiliki empat nilai:
  - **`0`**: Padding atas (top).
  - **`var(--a11y-collapse-padding-right, ...)`**: Padding kanan (right).
  - **`0`**: Padding bawah (bottom).
  - **`var(--a11y-collapse-padding-left, ...)`**: Padding kiri (left).
